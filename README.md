# Ostoslista

Ostoslista on tarkoitettu käyttäjän tarpeiden listaamiseen tulevia ostosreissuja varten. Käyttäjä kirjautuu 
sisään ohjelmaan omalla käyttäjätunnuksella ja salasanalla. Tämän jälkeen hän saa listan tulevista ostoksista.
Käyttäjä voi merkitä ostokset tehdyiksi tai lisätä listaan uusia tarpeita. Käyttäjä voi lisätä listaan eri kategorioita (esim. ruoka, hygienia, jne.), 
joiden alle ostoksia voi luokitella.

Sovellusta voi myös käyttää yhteisesti, jolloin ostoksia voidaan lisätä yhteiseen listaan esimerkiksi kämppisten kesken.
