# Ostoslista

Ostoslista on tarkoitettu käyttäjän tarpeiden listaamiseen tulevia ostosreissuja varten. Käyttäjä voi luoda käyttäjätunnuksen tunnuksenluomissivulla. Tämän jälkeen käyttäjä kirjautuu sisään ohjelmaan omalla käyttäjätunnuksella ja salasanalla. Kirjautumisen jälkeen hän näkee listan tulevista ostoksista. Jos lista on vielä tyhjä, niin käyttäjä voi lisätä listaan tuotteita. Jokaisella tuotteella tulee olla kategoria, joten käyttäjän tulee ensin varmistaa, että kategoria on jo olemassa tai luoda semmoinen.

Kategorian luomiseen on oma sivustonsa. Kirjautunut käyttäjä voi selata kategorioita ja lisätä uusia. Kategorioita voi poistaa vain, jos kategoriaan ei ole liitetty mitään tuotteita. Tämä selviää katsomalla listasta kategorian kokoa. Jos kategorian koko on nolla, niin poistaminen onnistuu. Muussa tapauksessa käyttäjän tulee poistaa ensin tuotteet ostoslistasta.

Käyttäjä voi merkitä ostokset tehdyiksi tai lisätä listaan uusia tarpeita. Myös tuotteiden poistaminen listasta on mahdollista. Käyttäjä voi lopuksi kirjautua ulos.
